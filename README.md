# Zehnder Multi Controller (Rainmaker)

Custom component wrapper for the Zehnder Multi Controller devices using the Rainmaker API.

This repository contains a Home Assistant custom component compatible with HACS. It depends on the `rainmaker-http` Python package for communicating with the Rainmaker API.

Installation
- Copy the `custom_components/zehnder_multi_controller` folder into your Home Assistant `config` directory or install via HACS once this repository is published.

Configuration
- Add integration via the UI and provide your Rainmaker host, username and password.
